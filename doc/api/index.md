# HTTP接口

## 简介

HellMapManager(HMM)支持以HTTP形式提供API服务，供有JSON处理和HTTP访问能力的MUD客户端调用。

开启HTTP服务需要在HMM主窗口的“接口”菜单栏里选择开启服务。

默认情况下，HMM会监听localhost的8466端口

在“接口”菜单栏内可以进行设置

* 服务端口 监听不同的端口，可以避免冲突或者开启不同的窗口进行监听
* 用户名和用户密码，开启后接口会进行HTTP Basic验证，即需要通过 http://用户名:用户密码@localhost:8466/的 形式进行调用。用户名或用户密码有任何一个为空则无需验证
* 打开程序时西东启动服务，勾选该选项后会在打开窗口后自动运行服务

成功启动服务后，主窗口下方的状态栏会提示 
```
接口服务正在监听8466端口
```
服务没有启动时，状态栏会提示
```
接口服务未启动
```
